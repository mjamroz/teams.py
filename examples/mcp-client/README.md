# Sample: MCP Client


### Available Commands

| Command | Description | Example Usage |
|---------|-------------|---------------|
| `agent <query>` | Use stateful ChatPrompt with MCP tools | `agent What's the weather like?` |
| `prompt <query>` | Use stateless ChatPrompt with MCP tools | `prompt Find information about Python` |
| `mcp info` | Show connected MCP servers and usage | `mcp info` |
| `<any message>` | Fallback to ChatPrompt with MCP tools | `Hello, can you help me?` |
